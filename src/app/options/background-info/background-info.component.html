Special Election Mode

You are playing a Special Mode!

Battle Ground Mode

In this mode, every state starts out tied. Scrabble to secure as many states as possible in 16 moves!

Remember that votes can change by as much as 3% on election day, so don't give up even if things look bleak.

32 Round Mode

This mode works the same as a standard game, but the game lasts for 32 rounds instead of 16 rounds.

See if the extra time can help you win big!

Randomized States Mode

In this mode, every state starts out with a random political preference. Try playing this mode a few times to see lots of interesting maps.

This is a great way to play with new strategies, as you will need to focus on different states each time you play.
