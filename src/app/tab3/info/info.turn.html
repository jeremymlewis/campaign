Instructions:

If these instructions still leave you with questions... try playing the game to see what happens.

Turn

On your turn you select 'Fundraise', 'Advertise', or 'Visit a State'. Each of these options will help you to win, so try them all! Use the map tab to decide which states to visit and advertise to.


After every 2 turns, you will draw an event card that will also affect the game with a random event.


The progress bar at the bottom is a great way to track how many turns are left.
